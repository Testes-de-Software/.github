# Sobre o Bitwarden

A <a href="https://bitwarden.com" target="_blank">Bitwarden</a> vislumbra um mundo onde ninguém seja hackeado, e reconhecemos que isso só é possível trabalhando juntos. A comunidade global do Bitwarden molda nossa missão de capacitar indivíduos, equipes e organizações a gerenciar com segurança suas informações confidenciais online.

#### Aqui, você pode:
<ul>
    <li>Revisar código + informações de lançamento/desenvolvimento de funcionalidades.
    <li>Propor Pull Requests (PRs).
    <li>Fazer Fork do Bitwarden.
</ul>

Para solicitar suporte, visite http://bitwarden.com/help e clique no botão "Contact support" no lado direito.

# Sobre os Repositórios Bitwarden

## Bitwarden Password Manager (Gerenciador de Senhas)

O Bitwarden Password Manager permite que empresas e indivíduos protejam seus dados online diante das crescentes ameaças de cibercrime. Use o Bitwarden Password Manager para gerar senhas fortes e exclusivas para cada conta que você usa online.

### Princípios de segurança em primeiro lugar
<ul>
    <li>Código aberto (Open source)
    <li>Criptografia de ponta a ponta (End-to-end encrypted)
    <li>Criptografia de conhecimento zero (Zero-knowledge)
    <li>Opção de auto-hospedagem (Self-host)
</ul>

#### Repositórios populares relacionados
<ul>
    <li><a href="https://github.com/bitwarden/server">server</a> - Infraestrutura/backend do Bitwarden (API, banco de dados, Docker, etc).
    <li><a href="https://github.com/bitwarden/clients">clients</a> - Aplicativos clientes do Bitwarden (web, extensão de navegador, desktop e CLI).
    <li><a href="https://github.com/bitwarden/android">android</a> - Aplicativos de Gerenciamento de Senhas e Autenticador para Android.
    <li><a href="https://github.com/bitwarden/ios">ios</a> - Aplicativos de Gerenciamento de Senhas e Autenticador para iOS.
</ul>
Saiba mais sobre o Bitwarden Password Manager lendo o <a href="https://bitwarden.com/help/bitwarden-security-white-paper/" target="_blank">Whitepaper de Segurança do Bitwarden</a>.

## Bitwarden Secrets Manager

O Bitwarden Secrets Manager permite que desenvolvedores, equipes de DevOps e de cibersegurança armazenem, gerenciem e implantem segredos (secrets) centralmente e em escala. Use o Secrets Manager para:
<ul>
    <li>Gerenciar e implantar segredos ao longo de todo o ciclo de vida de desenvolvimento.
    <li>Aumentar a produtividade e colaborar de forma segura.
    <li>Elevar o nível da sua proteção.
</ul>

#### Repositórios populares relacionados
<ul>
    <li><a href="https://github.com/bitwarden/sdk-sm">sdk-sm</a> - SDK do Bitwarden Secrets Manager.
    <li><a href="https://github.com/bitwarden/sm-action">sm-action</a> - GitHub Action para o Secrets Manager.
</ul>
Saiba mais visitando a <a href="https://bitwarden.com/help/secrets-manager-overview/" target="_blank">documentação do Secrets Manager</a>.

## Passwordless.dev

O Bitwarden Passwordless.dev fornece a estrutura de API que minimiza as complexidades associadas ao desenvolvimento de passkeys (chaves de acesso).
<ul>
    <li><b>Passkeys para seus usuários</b>: Para desenvolvedores que constroem aplicações, ofereça aos seus usuários finais uma experiência de login contínua, mais segura e que eles vão adorar. Mantenha seu fluxo de autenticação fluido e rápido para aumentar sua base de usuários.
    <li><b>Passkeys para sua força de trabalho</b>: Torne sua empresa à prova de phishing, permitindo que os membros da equipe acessem sistemas e aplicativos corporativos sem depender de senhas.
</ul>

#### Repositórios populares relacionados
<ul>
    <li><a href="https://github.com/bitwarden/passwordless-server">passwordless-server</a> - Infraestrutura/backend do Passwordless.dev (API, banco de dados, Docker, etc).
    <li><a href="https://github.com/bitwarden/passwordless-react-example">passwordless-react-example</a> - Exemplo de uso do Passwordless.dev com React.
</ul>
Saiba mais visitando a <a href="https://docs.passwordless.dev/" target="_blank">documentação do Passwordless.dev</a>.
